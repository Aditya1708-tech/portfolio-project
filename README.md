## ✨ Features & Highlights

### 🎨 Visuals & Aesthetics
* **Dynamic Backdrop Spotlight:** A high-performance radial spotlight glow that tracks cursor movements across the page.
* **Custom Cursor Follower:** A dual-element custom trailing cursor (`dot` & `ring`) with context-aware scaling and hover styles for links, buttons, and cards.
* **Modern Typography & Glassmorphism:** Clean, modern layouts styled using **Tailwind CSS v4** combined with backdrop filters, subtle gradients, and border-glow highlights.

### ⚡ Interactive Animations
* **GSAP & ScrollTrigger Integrations:**
  * **Text Splitting & Reveal:** Character-by-character text animations for hero headers.
  * **Interactive Stat Counters:** Dynamic rolling stat counters triggered on scroll.
  * **Drawing Path Timelines:** Experience connectors draw themselves dynamically as the user scrolls.
* **Framer Motion Micro-animations:** Smooth navigation menu transitions and modal animations.
* **Magnetic Hover Effects:** CTA buttons dynamically attract to the cursor within proximity.
* **3D Tilt Cards:** Premium tilt mechanics on project cards and achievements.

### 🏆 Academic & Career Showcase
* **Google TechSprint Finalist Node:** Displays detail regarding being in the **Top 10 Nationwide Finalists** of the *Build with Google TechSprint* challenge.
* **Interactive Lightbox Certification Viewer:** Users can view and download high-resolution certificates (images & PDFs) directly through a modern lightbox overlay.
* **Contact Messenger Panel:** Fully functional glassmorphic contact form featuring validation, submission states, and feedback loops.

## 🛠️ Tech Stack & Libraries

### Core Technologies
* **React 19** - Component-based architecture.
* **TypeScript** - Strict type safety.
* **Vite** - Lightning-fast build tool and dev server.

### Styling & Animation Engines
* **Tailwind CSS v4** - Performance-oriented utility CSS framework.
* **GSAP (GreenSock Animation Platform)** - Timeline animations and ScrollTrigger tracking.
* **Framer Motion** - Orchestrated entry/exit animation loops.
* **Lucide React** - Vector iconography.

## ⚙️ Local Development Setup

To run this project locally, ensure you have **Node.js** installed, then execute the following steps:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Aditya1708-tech/portfolio-project.git
   cd portfolio-project
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Launch Local Development Server:**
   ```bash
   npm run dev
   ```
   *The application will launch on your default port (usually `http://localhost:5173`).*

4. **Build Production Assets:**
   ```bash
   npm run build
   ```

5. **Preview Production Build Locally:**
   ```bash
   npm run preview
   ```
